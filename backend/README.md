# DECIMETER Backend

Production-ready backend for DECIMETER clothing brand e-commerce platform.

## Features
- User authentication with JWT
- Product management
- Shopping cart functionality
- Order processing and checkout
- Admin endpoints for product/order management

## Tech Stack
- Python 3.13.4
- Flask
- PostgreSQL (via Render)
- SQLAlchemy ORM
- Flask-JWT-Extended
- psycopg v3 (compatible with Python 3.13.4)
- Gunicorn

## Deployment on Render

### 1. Prerequisites
- Render account
- PostgreSQL database created on Render
- Frontend deployed on Render
